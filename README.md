# Тестовое задание: Online Courses

Это веб-приложение для просмотра онлайн-курсов. Пользователи могут просматривать курсы по категориям, искать по названию или автору, и загружать больше курсов.
Выполнено по макету с соблюдением Pixel-Perfect 
[Макет](https://www.figma.com/design/iuXLXifXgv4ZCzad0KeYKr/Test-Task?node-id=2249-2298&t=sGOczqEiidD9VV8x-0)

**Демо:** [https://online-courses-ochre.vercel.app](https://online-courses-ochre.vercel.app)

## Функции

-   Просмотр списка курсов
-   Фильтрация по категориям
-   Поиск по названию или автору
-   Адаптивный дизайн для мобильных устройств
-   SEO-оптимизация

## Установка

1. Клонируйте репозиторий:

    ```
    git clone https://github.com/kimmito/online-courses.git
    cd online-courses
    ```

2. Установите зависимости:
    ```
    npm install
    ```

## Запуск

Запустите локальный сервер разработки:

```
npm run dev
```

Откройте браузер и перейдите по адресу `http://localhost:3000`.

## Сборка

Для продакшена:

```
npm run build
```
